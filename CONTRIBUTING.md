# 🤝 Contributing to VPN Network Reset

Thank you for your interest in contributing!

## How to Contribute

### 🐛 Bug Reports
Please include:
- Windows version
- VPN client used
- Error message (if any)
- Steps to reproduce

### 💡 Feature Requests
- Clear description of the feature
- Use case / why it's needed
- Any relevant links or research

### 🔧 Pull Requests
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test on Windows 10/11
5. Commit with clear message: `git commit -m "Add: your feature"`
6. Push: `git push origin feature/your-feature`
7. Open a Pull Request

### 📋 Coding Standards
- Use PowerShell 5.1+ compatible syntax
- Test with `-ErrorAction Stop` for critical operations
- Include comments for non-obvious code
- Keep functions small and focused

## Questions?
Open an Issue for discussion!
