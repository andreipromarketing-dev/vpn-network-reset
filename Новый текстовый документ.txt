Чтобы увеличить скорость интернета на Windows 10, можно попробовать несколько способов, связанных с настройками системы, роутера и сетевого оборудования. Важно учитывать, что иногда проблема может быть не на стороне пользователя, а у провайдера. 

Настройки Windows 10
Отключите функцию «Автонастройка окна» (Auto-Tuning). По умолчанию Windows резервирует часть пропускной способности для системных нужд, что может снижать скорость. Чтобы отключить эту функцию:

Нажмите Win + R, введите cmd и откройте командную строку от имени администратора.
Введите команду: netsh interface tcp set global autotuninglevel=disabled.
Перезагрузите компьютер. 
drivereasy.com +1
Увеличьте кэш DNS. Это ускорит загрузку часто посещаемых сайтов. Для изменения настроек:

Нажмите Win + R и введите regedit, чтобы открыть редактор реестра.
Перейдите в раздел HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\DNScache\Parameters.
В этом разделе создайте или измените параметры, увеличив значения для кэша. 
Сбросьте настройки сети. Это можно сделать с помощью команд в командной строке (от имени администратора):

ipconfig /flushdns — очищает кэш DNS.
netsh winsock reset — сбрасывает настройки Winsock.
netsh int ip reset — сбрасывает параметры IP. 
После выполнения команд перезагрузите компьютер.

Отключите ограничение резервируемой пропускной способности. По умолчанию Windows резервирует 20% пропускной способности для системных нужд. Чтобы снять ограничение:

Нажмите Win + R, введите gpedit.msc и откройте редактор локальной групповой политики.
Перейдите в раздел: «Конфигурация компьютера» → «Административные шаблоны» → «Сеть» → «Планировщик пакетов QoS».
Включите опцию «Ограничить резервируемую пропускную способность» и установите значение 0%.

Если в системе отсутствует gpedit.msc, можно использовать программы вроде Winaero Tweaker для аналогичных настроек. 

Настройки сетевого оборудования
Обновите драйверы сетевой карты. Устаревшие драйверы могут снижать скорость передачи данных. Скачайте актуальные версии с сайта производителя материнской платы или ноутбука, либо обновите их через «Диспетчер устройств» в Windows. 
hi-tech.mail.ru +1

Проверьте кабель и подключение. При проводном подключении убедитесь, что кабель не повреждён и правильно обжат. Используйте 8-жильный кабель, если требуется скорость выше 100 Мбит/с. Попробуйте подключить интернет напрямую от провайдера к компьютеру, минуя роутер, чтобы проверить, в чём проблема. 
help-wifi.com +1

Настройте параметры сетевой карты. В «Диспетчере устройств» откройте свойства сетевой карты, перейдите на вкладку «Дополнительно» и проверьте параметры. Можно отключить режим энергосбережения (Power Saving Mode) и настроить «Скорость и дуплекс» (например, установить 1.0 Gbps Full Duplex вместо автоматического согласования). 
hi-tech.mail.ru +1

Настройки роутера
Уберите лишние устройства из сети. Проверьте список подключённых устройств в настройках роутера и отключите ненужные — они создают нагрузку на сеть. 

Смените канал Wi-Fi. По умолчанию роутер автоматически выбирает канал, но в многоквартирных домах он может быть перегружен. Используйте программы вроде WiFi Analyzer, чтобы определить менее загруженный канал, и вручную установите его в настройках роутера. 
hi-tech.mail.ru +1

Обновите прошивку роутера. Устаревшая прошивка может снижать скорость соединения. Проверьте наличие обновлений на сайте производителя и установите их через веб-интерфейс устройства. 

Настройте QoS (Quality of Service). Эта функция позволяет распределять трафик так, чтобы важные приложения (игры, видеозвонки) получали приоритет. Настройки QoS обычно находятся в разделе «Приоритизация данных» или «Bandwidth Control» в веб-конфигураторе роутера. 

Дополнительные рекомендации
Отключите VPN, если он используется. VPN-серверы могут значительно снижать скорость соединения. 
Проверьте систему на наличие вредоносного ПО. Вирусы и шпионские программы могут использовать интернет-трафик в фоновом режиме. 
unisender.com +1
Ограничьте фоновые приложения. Закройте программы, которые активно используют трафик (торрент-клиенты, облачную синхронизацию и т. д.). 
Используйте турбо-режимы в браузерах. В Яндекс Браузере и Opera есть функция сжатия данных, которая ускоряет загрузку страниц. 
Если после выполнения этих действий скорость не улучшилась, стоит обратиться к провайдеру для проверки состояния линии и тарифа.
